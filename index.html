둘러보기

웹 어플리케이션

단순히 정보를 보여주는 홈페이지가 아닌 전문 플랫폼 중심의 웹을 구현을 목표로 합니다.

모바일 어플리케이션

IOS, ANDROID 동시에 개발하며 휴대폰 인증, 결제, GPS, 알림 등등 반응형 모바일 앱으로 구현하지 못하는 기능을 구현합니다.

디자인

유저 프렌들리한 UI/UX로 사용자가 쉽게 다가갈 수 있는 디자인으로 구현합니다.

틀에 갇혀 있지 않는 어플리케이션을 제작하기 위해 프로젝트의 기획, 디자인을 도와드립니다.

웹, 애플의 앱스토어, 구글의 플레이 스토어, 에는 수 많은 웹사이트들과 어플리케이션이 존재합니다. 하지만 이중 실제로 사용자들이 자주 사용하는 어플리케이션은 몇 안됩니다.

저희는 고객님의 프로젝트가 성공할 수 있도록 기획부터 시작해, 개발, 디자인 과정을 고객님과 상의하면서 진행합니다.

저희가 가지고 있는 기술

4_ds는 어플리케이션의 최적화를 위해 현재 세계에서 가장 많이 사용되고 있는 프론트엔드, 벡엔드 개발 프레임워크를 사용하고 있습니다.

React 90%
React Native 95%
MySQL & mongoDB 95%
Node.js 98%
